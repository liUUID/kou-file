## 📚 项目技术栈

后端：

- 核心框架：Spring Boot 2.6.6
- orm: MyBatis Plus
- 数据库：MySQL 8.0
- 权限安全控制：Sa-Token
- 本地缓存：Ehcache
- 文件上传：本地、七牛云、阿里云OSS
- 第三方登录：JustAuth
- 模板渲染：Thymeleaf
-
前端：

- Layui v2.5.5
- Jquery
- Ajax
